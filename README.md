# 대전 트래블 라운지.
팀 프로젝트(총 3인)로 개발한 대전 트래블 라운지(이하 "라운지") 웹사이트입니다. 
<br/>
실제 라운지에서 제공하는 체험프로그램, 관광기념품(이하 "상품") 판매기능을 구현하였습니다.

개발언어: Java, Jsp, HTML/CSS/JS
<br/>
개발도구: Eclipse, Oracle Sql Developer, Spring, Spring Sequrity,
